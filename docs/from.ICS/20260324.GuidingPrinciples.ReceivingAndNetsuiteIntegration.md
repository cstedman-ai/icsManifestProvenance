# 20260324 — Guiding Principles — Receiving and NetSuite Integration

**Contributors:** Liam Jones, Frank D'Arrigo, Corey Stedman

---

## Guiding Principles

### 1. QR-Code-First Receiving

All vendor shipments must include a scannable QR code that encodes the full packing list. Scanning the code on a phone or handheld device must be sufficient to pull the structured manifest into our system — no manual data entry, no paper transcription.

### 2. Vendor Contract Revision

Existing vendor contracts must be revised to require QR-code-based packing documentation as a condition of shipment. New vendor onboarding must include this requirement from day one so adoption is universal, not opt-in.

### 3. NetSuite as the System of Record

Receiving data must flow into NetSuite. When a QR code is scanned, the resulting manifest data should be appended to the corresponding Item Receipt (IR) in NetSuite via its API. The procurement ticket, the packing list, and the receiving record must all converge in one place.

### 4. Mobile-Native Scanning

Warehouse staff must be able to scan and receive using a phone or tablet. If a SOC/SOX-compliant scanning tool already exists for NetSuite, adopt it. If not, we build the capability ourselves within this application and interface with NetSuite's API.

### 5. Line-Item Verification at Scan Time

The scanned packing slip must present interactive checkboxes per line item so the receiver can confirm each item as received (or flag it as missing/damaged) in real time. This produces a verified, item-level receipt — not a bulk acknowledgment.

### 6. SOC/SOX Compliance by Design

Every scan, checkbox confirmation, and submission must be timestamped and attributed to a specific user. The audit trail must satisfy SOC and SOX requirements without relying on manual signatures or paper records.

### 7. Automatic Reconciliation

The system must auto-compare the scanned/submitted manifest against the original purchase order and surface discrepancies (quantity deltas, missing serials, unexpected items) immediately — not after a manual review cycle.
